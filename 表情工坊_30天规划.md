#### 上下文推荐的详细设计

**UI位置**：主界面右侧增加"智能推荐"侧边栏

**使用流程**：
1. 用户粘贴/输入一段聊天上下文（如"今天考试挂了，好难过"）
2. 系统分析文本，提取情绪（悲伤）+ 关键词（考试、挂科）
3. **模式A-库内推荐**：在已有表情库中搜索标签含"安慰""考试""难过"的表情，展示Top-3
4. **模式B-AI生成**：若库内匹配度低，自动生成新表情（如"一只猫说：下次一定行"）

**技术实现（分层）**：
- **保底方案**：jieba分词 → 匹配标签 → 排序返回
- **进阶方案**：调用大模型API（通义千问/DeepSeek）分析情绪+生成推荐标签
- **展示**：推荐结果以缩略图列表呈现，点击直接复制/拖拽使用

**面向对象设计示例**：
```python
# controllers/recommend_controller.py
class RecommendController:
    def __init__(self, db_service, ai_service):
        self.db = db_service
        self.ai = ai_service

    def recommend_from_library(self, context: str, top_k: int = 3):
        # 基于上下文从库内推荐
        keywords = self._extract_keywords(context)
        emotions = self._analyze_emotion(context)
        return self.db.search_by_tags_and_keywords(keywords, emotions, top_k)

    def generate_for_context(self, context: str, style: str = "搞笑"):
        # 基于上下文AI生成新表情
        prompt = self._build_prompt(context, style)
        image_path = self.ai.generate_image(prompt)
        return self._save_to_library(image_path, auto_tag=["AI生成", style])
```

## 四、数据库表结构设计

```sql
-- images: 表情包主表
CREATE TABLE images (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    file_path TEXT NOT NULL UNIQUE,
    file_type TEXT NOT NULL,
    file_size INTEGER,
    width INTEGER,
    height INTEGER,
    thumbnail_path TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- tags: 标签表
CREATE TABLE tags (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL UNIQUE,
    color TEXT DEFAULT '#FF6B6B',
    category TEXT
);

-- image_tags: 多对多关联
CREATE TABLE image_tags (
    image_id INTEGER REFERENCES images(id) ON DELETE CASCADE,
    tag_id INTEGER REFERENCES tags(id) ON DELETE CASCADE,
    PRIMARY KEY (image_id, tag_id)
);

-- usage_history: 使用记录（用于统计和推荐）
CREATE TABLE usage_history (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    image_id INTEGER REFERENCES images(id),
    used_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    context TEXT,
    source TEXT DEFAULT 'clipboard'
);

-- ai_generations: AI生成记录
CREATE TABLE ai_generations (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    image_id INTEGER REFERENCES images(id),
    prompt TEXT,
    style TEXT,
    generated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
